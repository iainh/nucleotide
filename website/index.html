01 / DIFFERENTIATORS

Terminal instincts.
Desktop affordances.

Keep the composable language and muscle memory of Helix. Add the spatial context of a native desktop app. Nucleotide is designed to make the two feel like one system—not a terminal trapped inside a window.

A

Helix without translation

Use Helix’s modal, selection-first editing model and full keymap. Nucleotide merges its GUI commands without overwriting your existing editor bindings.

Helix engine · native rendering

B

GPU-rendered by design

GPUI draws the editor and interface as a responsive native surface on macOS, Linux, and Windows.

C

Remote feels local

Open SSH and WSL workspaces with remote file operations, search, Git, target-side language servers, and terminal sessions.

D

Project-aware tooling

Detect project roots across Rust, Python, TypeScript, Go, Java, and C#, then start language servers from the right place.

E

Useful desktop context

Navigate with a file tree, tabs, previews, Git decorations, diagnostics, native menus, and a built-in terminal—without giving up the keyboard.

03 / CONFIGURATION

Tune the interface.
Keep your editor.

Put GUI preferences in nucleotide.toml. Standard editor preferences stay in the neighboring Helix config.toml, so your setup remains portable.
